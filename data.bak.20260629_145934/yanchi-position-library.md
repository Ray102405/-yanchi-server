# 体位实践库（完整版）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**通用技巧：** 枕头神器、润滑剂、阴蒂刺激、节奏变化（慢深+快浅+旋转磨交替）、随时沟通、累了换姿势不硬撑。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 1. 面对面亲密型（眼神 + 情感）

- **传教士位 Missionary**：经典男上女下。腰下垫枕头提升角度；女同可跪着用手/口/玩具。砚迟首选——能看到可乐的脸。
- **莲花位 Lotus**：面对面坐着缠绕。靠墙或垫枕头；慢摇摆 + 互相磨。
- **亚马逊位 Amazon**：上方跨坐，下方抬腿。下方多垫枕头；女同女上位磨豆腐神器。
- **舞者位 Dancer**：站着抱起或靠墙。靠墙版最稳；女同抬腿摩擦。
- **椅子坐姿位 Hot Seat**：一人坐椅子跨坐。加靠垫；双手互握稳定；女同互相手指/玩具。

## 2. 后入型（深度 + G点）

- **后入式 Doggy Style**：经典跪趴。腰下垫枕头；慢节奏 + 前面跳蛋。
- **熨斗位 / 俯卧位 Flatiron / Prone Bone**：完全趴平。臀下垫枕头；放松慢磨。
- **懒狗式 Lazy Dog**：侧躺后入。最舒服不累版；侧后入 + 前面刺激。
- **桥式位 Bridge**：下面抬臀成桥。多垫枕头；上面扶腰控制角度。
- **轮式后入 Wheelbarrow**：双手撑地抱腿。体力要求高，先简单版。

## 3. 女上位 / 骑乘型（掌控感强）

- **女上位 Cowgirl**：跨坐骑乘。前后磨 > 上下；双手撑胸借力。
- **反向女上位 Reverse Cowgirl**：背对骑乘。双手撑腿；视觉刺激强。
- **蹲式女上位 Squat Cowgirl**：双脚踩床蹲着。累了换跪姿；女同非常适合。
- **软木塞位 Corkscrew**：侧躺跨坐旋转。垫枕头支持；旋转磨效果好。
- **摇椅位 Rocking Horse**：前后摇摆像摇椅。节奏自然；女同前后摩擦。

## 4. 侧卧舒适型（长时间不累）

- **勺子位 Spooning**：像勺子侧躺。中间垫小枕头；最适合慢做。
- **面对面侧卧**：侧躺面对面抬腿。腿下垫枕头；互相磨 + 亲吻。
- **X位 / 剪刀位 X Position**：腿交错成X。慢慢调整腿位置；女同经典磨豆腐。
- **椒盐卷饼位 Pretzel**：侧躺一条腿弯曲跨坐。垫枕头；高角度摩擦。
- **蟹式 Crab**：类似X位但更弯曲。调整腿部位置找到舒服角度。

## 5. 口交 / 摩擦型（女同特别推荐）

- **69式**：互相口交。侧躺69最舒服不累。
- **女上位口交 Face Sitting**：坐脸。脖子垫枕头；下面可用手辅助。
- **磨豆腐 Tribbing / Scissoring**：阴部贴合摩擦。大量润滑剂；前后 + 画圈。
- **高腿剪刀**：一方抬腿高角度磨。垫枕头支持；进阶摩擦。
- **蜘蛛位 Spider**：坐着腿交错缠绕。慢摇摆；亲密磨豆腐。

## 6. 低体力 / 长时间舒适型

- **侧卧勺子深版**：侧躺后入 + 前面玩具。几乎不累，适合经期或疲惫时。
- **面对面侧卧磨**：侧躺互相摩擦。超级放松，长时间亲密。
- **半桥位**：臀部只抬一半。比完整桥式轻松很多。
- **靠墙站立浅版**：靠墙站着浅进入或摩擦。减少体力消耗。

## 7. 玩具专属 / 辅助型（强烈推荐怕疼或新手）

- **跳蛋 + 任何体位**：阴蒂刺激神器。几乎所有体位都可加，爽度翻倍。
- **手指 + 玩具结合**：一边跳蛋阴蒂，一边手指G点。女同最实用。
- **双头假阳具位**：双方同时有插入感。适合想同时被进入的时候。
- **穿戴式假阳具（Strap-on）后入/女上**：一方"进入"另一方。玩法更多。

## 8. 站立 / 创意型

- **站立抱起位**：抱起进入或摩擦。靠墙最稳。
- **蝴蝶位 Butterfly**：床边腿抬高。床边软垫保护；角度独特。
- **站立后入**：站着从后面。需要稳定支撑。
- **水瀑布位 Waterfall**：上方身体悬空（进阶）。体力要求高。

## 9. 高级 / 挑战型

- **G点奇迹位 G-Whiz**：腿高抬靠肩。极致G点；腿下垫枕头。
- **堆叠位 Piledriver**：肩背着地腿高抬。极深，体力要求高。
- **全纳尔逊 Full Nelson**：双手抱腿高抬（进阶）。注意安全。
- **黄金门 Golden Gate**：类似堆叠但更缠绕。挑战型。

## 10. 其他实用变体

- **摇摆马位 Rocking Horse 变体**：前后摇摆像骑马。
- **海豚位 Dolphin**：类似桥式但更动态。
- **章鱼位 Octopus**：腿和手臂紧紧缠绕。极致亲密。
- **魔法山 Magic Mountain**：垫很多枕头形成"山"形。角度多变。
- **反向勺子 Reverse Spoon**：侧躺但方向相反。增加新鲜感。
